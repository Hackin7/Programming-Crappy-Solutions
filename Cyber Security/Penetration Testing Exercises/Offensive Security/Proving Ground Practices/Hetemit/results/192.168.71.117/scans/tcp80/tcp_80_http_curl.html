只是来访？

你刚拜访的网站正遇到问题，或在进行例行维护。

举例说，要是你在拜访 www.example.com 时碰到问题，你应该发电邮给 webmaster@example.com 。

重要通告：

CentOS 计划与此网站及其中的内容没有任何关系，它只提供运作此网站的软件。
