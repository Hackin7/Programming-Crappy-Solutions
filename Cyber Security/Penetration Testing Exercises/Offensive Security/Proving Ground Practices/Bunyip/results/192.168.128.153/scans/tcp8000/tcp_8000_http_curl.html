s3cur3 r3pl

Welcome to s3cur3 r3pl, a cryptographically secured node.js REPL. s3cur3 r3pl enables its users to execute code as well as allow third parties to execute approved code blocks on their behalf.

To run code it must first be signed with the user's API key. The signature and the code are then submitted to the server for execution. To sign the code it is appended to the your API key and then hashed using MD5 as follows: 

signature = md5(<API-KEY>|CODE)

We've prefilled the REPL with code signed with our own key to demonstrate just how secure our REPL is!
